# 🌍 Language Detection using Machine Learning

## 📌 Project Overview
The Language Detection Project is a Machine Learning and Natural Language Processing (NLP) based application that detects the language of a given text input. The system uses text preprocessing, feature extraction, and a trained classification model to predict the language accurately.

This project was developed as part of an academic capstone project and demonstrates practical implementation of NLP, Machine Learning, and data visualization concepts.

## 🚀 Features
- Language detection from user input text
- Text preprocessing and cleaning
- Feature extraction using CountVectorizer
- Naive Bayes classification model
- GUI interface using Tkinter
- Model evaluation and prediction
- Option to search detected language information
- Data visualization using Matplotlib and Seaborn

## 🧠 Machine Learning Workflow
The project follows a standard ML pipeline:

1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Language Prediction
